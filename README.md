# webpack-pug

## Quick start
```bash
npm install
```

### Development
```bash
npm start
```

### Production
```bash
npm run build
```

## Add a new page
- Add a new field in *app/pages/config.json*, for example:
```json
{
  "name": "aboutus",
  "url": "about-us"
}
```
- Create a file and its directories: *app/pages/aboutus/js/index.js*
